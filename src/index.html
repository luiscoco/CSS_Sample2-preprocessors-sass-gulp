SCSS

General syntax

SCSS aims to be a superset of CSS, so the syntax is the same with some additions like variables, modules, mixins etr

Curly brackets, colons and semicolons are used like in CSS

Variables

SASS variables

Sass variables exist only in compile time. Sass variables are compliled to 'fixed' values so they do not exist as variables in runtime

Hyphens and underscore in variable names are treated as the same symbol, so $font-size and $font_size will be the same

CSS variables

CSS variables exist in runtime. So we can use them for example inside media queries so that viewport width will affect their value
